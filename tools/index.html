Vanilla JavaScript

A list of sample using plain HTML/JavaScript to operate on JACDAC devices, built with this library.

  • packets, sniff all packets on the bus

  • devices, list of connected devices

  • speed test, test end-to-end speed of WebUSB-JACDAC connection

  • console, turns on verbose console and display messages

  • streaming, live streaming of sensor data into range widgets

  • streaming-rickshaw, live streaming of sensor data into graphs

  • flashing, flash new firmware onto a device

  • graphql, querying against the JacDac graphql instance

  • graphql-schema, query graphql schema
