## Collecting traces

### saleae

Use [Saleae LOGIC](https://www.saleae.com/) analyser to record trace on the JACDAC data line.

* Add an ``Async Serial`` protocol analyser
* Select ``1Mbit``
* Export to ``CSV/hex``.
